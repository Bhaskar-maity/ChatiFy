## Installation:
  #### Install NodeJS
  #### Clone the repository
  ```bash
  git clone https://github.com/Bhaskar-maity/ChatiFy.git
  ```
  ### `Go to /server/ folder`
  #### Install dependencies
  ```bash
  npm install
  ```


  #### Start the server (run server.js file)
  ```bash
  npm start
  ```
  ### `Go to /client/ folder`
  In the project directory, you can run:
  #### Install dependencies
  ```bash
  npm install
  ```
  #### Start the server
```bash
npm start
```
